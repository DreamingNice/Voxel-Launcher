# Voxel Launcher
The Voxel Launcher is java script based a minimal launcher for Minecraft. The project still on devlopment state so there can be little bugs.

## IMPORTANT
To use this launcher you need at least java 17 or below. Java 8 is unsupported for now!

## Installation
For now there is only one version of the launcher is released so you can download and install useing the installer.

The launcher only supports Windows for now.

## NPM AND NODE
The launcher needs Node 18+
If you are cloning the project you need to install npm to root of the project.
